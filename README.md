# EventTrack — Сайт учёта мероприятий образовательного учреждения

Учебный проект для демонстрации работы с GitHub: простая система учёта мероприятий.

## Функционал

* Добавление новых мероприятий (название, дата, тип)
* Просмотр списка всех мероприятий в таблице
* Фильтрация по дате и типу мероприятия
* Удаление мероприятий

## Технологии

* HTML5 — структура страницы
* CSS3 — стилизация интерфейса
* JavaScript (чистый) — логика работы с данными
